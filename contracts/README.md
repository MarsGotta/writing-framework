# Contratos (puntero)

La única fuente editable de contratos y schemas es **`writeonmars/contracts/`**
(es la copia que viaja con el preset). Este directorio quedó como puntero el
2026-07-04 para eliminar el riesgo de divergencia entre espejos: edita siempre
la copia del preset.
